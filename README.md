# iso27001-iam-governance

Projeto de estudo independente sobre Gestão de Identidades e Acessos (IAM) baseado nos controles públicos da ISO/IEC 27001:2022.

**GitHub Pages:** [Documentação do projeto:](https://gilbertocrv.github.io/iso27001-iam-governance/index.html)

---

## Estrutura

```
iso27001-iam-governance/
│
├── index.html                          # Página principal — mapa de controles
│
├── controles/                          # Deep dives por controle ISO 27001
│   ├── 5.15.html                       # Access Control Policy
│   ├── 5.16.html                       # Identity Management
│   ├── 5.17.html                       # Authentication Information
│   ├── 5.18.html                       # Access Rights
│   ├── 8.2.html                        # Privileged Access Rights (em breve)
│   ├── 8.3.html                        # Information Access Restriction (em breve)
│   └── 8.5.html                        # Secure Authentication (em breve)
│
├── ferramentas/                        # Ferramentas interativas
│   ├── policy-builder.html             # Gerador de rascunho de política
│   └── sod-simulator.html             # Simulador de conflitos SoD
│
├── policies/                           # Documentos de referência (Markdown)
│   └── Access_Control_Policy.md
│
├── _config.yml                         # Configuração GitHub Pages
└── README.md                           # Este arquivo
```

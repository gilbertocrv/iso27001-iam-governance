Importante / Disclaimer

Este artefato é educacional. Revise os comandos no Visual Studio Code antes de executar. Requer módulo Microsoft.Graph e permissões de escrita no tenant. O atributo UsageLocation foi incluído como boa prática comum em cenários de identidade e licenciamento no ecossistema Microsoft.

Glossário Técnico
UPN (User Principal Name) Login principal do usuário, normalmente no formato user@domain.com.
Connect-MgGraph Comando de autenticação via Microsoft Graph PowerShell SDK.
ForceChangePasswordNextSignIn Exige redefinição de senha no primeiro acesso.
UsageLocation Atributo frequentemente necessário para cenários de licenciamento e conformidade regional.

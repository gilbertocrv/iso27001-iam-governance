Instruções do Template

O arquivo CSV deve ser estruturado com os cabeçalhos exatos conforme o exemplo abaixo:

Nome UPN Cargo Departamento
Exemplo Usuario usuario@dominio.com Analista de IAM Segurança da Informação

⚠️ Nota de Segurança

Os scripts gerados configuram ForceChangePasswordNextSignIn como True por padrão, garantindo que a conformidade com a política de troca de senha no primeiro acesso seja mantida.
